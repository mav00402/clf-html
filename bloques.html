Elementos block y elementos inline

Un elemento block siempre comienza en una nueva linea, y los navegadores automaticamente agregan algo de espacio (un margen) antes y despues de cada elemento.
Un elemento bloque simpre toma el total del ancho disponible
dos bloques normalmente utilizado son "p" y "div"
el elemento "p" define un parrafo en el documento html
el elemnto "div" define una division o seccion en el documento

El elemnto div

El elemento "div" se suele utilizar como contenedor para otros elemento html
Este mismo no tiene atributos requeridos pero son normales los atributos style, class e id
Cuando se utilizan en conjunto con CSS, el elemento div puede ser usado para dar formato al bloque del contenido
